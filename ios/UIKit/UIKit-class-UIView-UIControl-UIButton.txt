An instance of the UIButton class implements a button on the touch screen. 
A button intercepts touch events and sends an action message to a target object when tapped. 

Methods for setting the target and action are inherited from UIControl. 
This class provides methods for setting the title, image, and other appearance properties of a button. 
By using these accessors, you can specify a different appearance for each button state.
